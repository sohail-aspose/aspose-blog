#### Aspose と Perpetuum Software 社とのパートナーシップ

Silverlight Viewer for Reporting Services は、Silverlight アプリケーションで Reporting Services レポートの View 機能を提供するマーケットで最初の製品です。どの Silverlight アプリケーションにも簡単に統合できるネイティブ Silverlight コントロールです。クラシカルなデスクトップ アプリケーションで利用可能な機能のセット (ダイナミック スケーリング、アニメーションのスムーズなパニング、インタラクティブなソート機能、検索機能などを含む) を使用します。

Aspose と Perpetuum Software 社とのパートナーシップを記念して、Perpetuum 社では、Aspose 製品のお客様に、2011年 1月、2月の間、Silverlight Viewer for Reporting Services を 25% Off で提供いたします。 [キャンペーンのお申し込みは、Aspose まで email にてご連絡ください。][4]

#### 製品スポットライト – Aspose.Total for SSRS

[](http://www.aspose.com/community/files/52/ssrs-rendering-extensions/aspose.total-for-reporting-services/default.aspx?utm_source=newsletterJan11&utm_medium=email&utm_campaign=download)Microsoft SQL Server では、開発者に特別なチャレンジを提供します。レポートを一般的なビジネス ドキュメント フォーマットに確実に変換するのに役立つように、Aspose では、Reporting Services 用にレンダリング エクステンションを開発しました。Reporting Services 用のすべてのエクステンションのトータル パッケージでは、Word 文書、Excel スプレッドシート、プレゼンテーション フォーマットと操作、またバーコードのレンダリング、PDF ファイルの保存、SWF や FLV ファイルのレポートなど堅牢なエクステンションを提供します。

Aspose.Total for SSRS には、Aspose のすべての Reporting Services 製品、Aspose.Words、Aspose.Slides、Aspose.Cells、Aspose.Pdf、Aspose.BarCode および Aspose.Flash for Reporting Services が含まれます。多くのエクステンションは、Microsoft SQL Server 2000、2005 および 2008 Reporting Services と Report Viewer で動作します。([ここ詳細は、から参照可能です。][5]

#### 技術記事 – Dataset ベースのレポートでバーコードを表示

ドキュメント管理でバーコードをよく使用します。Aspose.BarCode for SSRS は、バーコードをレポートへ追加する機能を含む SSR の機能を拡張します。本技術記事では、Dataset ベースのレポートでバーコードを簡単に表示する方法を紹介します。記事全文は、ここから参照可能です。

#### チュートリアル ビデオ – Aspose.Cells for SSRS と Visual Studio 2005 または 2008 Report Designer の統合

Aspose.Cells では、SSRS からネイティブな MS Excel レポートを出力することができます。Microsoft Excel のオートメーションを必要とせず、アプリケーション内からファイルの読み込み、書き込みおよび操作ができます。

本ビデオ チュートリアルでは、Aspose.Cells for SSRS と Visual Studio 2005 または 2008 Report Designer を手動で統合する方法を紹介します。チュートリアルでは、2分以内で設定できる方法を紹介します。ビデオはここから参照可能です。

#### 移植のヒント – Aspose.Slides を使用して、プレゼンテーションのアニメーションにピクチャ フレームを追加

Aspose.Slides for .NET では、プログラム上でプレゼンテーションを操作し、イメージ、フレームおよびアニメーションの追加ができます。プログラム コードは綺麗で効率的で読みやすいです。本ヒントでは、Visual Studio Tools for Office Automation 2008 を使用した場合と Aspose.Slides を使用した場合を比較して、PowerPoint プレゼンテーションにフレーム付きのピクチャを配置したりアニメーションを追加する方法を紹介します。Aspose.Slides を使用した方が簡単だと分かるかと思われます。本ヒントは、ここから参照可能です。
