![Aspose.Slides icon][1]

We're please to share the announcement of Aspose.Slides for .NET 6.3.0. We have added some new features in this new release that include accessing document properties of password protected presentation and support for embedded videos support in case of PPTX. We have  also added the long awaited, exporting to SVG image support for PPTX Slides support in this release. Many presentation rendering issues including increased PDF size and improper rendering has been resolved.  Some of presentation access issues along with the data access from added Ole frame has also been resolved in this new release. The chart support has also been enhanced in this new release and support for adding formatting for chart titles and labels has also been introduced in this new release.

To see a complete list of fixes and to download Aspose.Slides for Java 6.3.0, [please go to the download page][2].
