#### Product Spotlight

[![][1]](http://www.aspose.com/community/files/67/jasperreports-exporters/aspose.words-for-jasperreports/default.aspx)[Aspose.Words for JasperReports][2] is the only known solution on the market that makes possible exporting reports from JasperReports and JasperServer to Microsoft Word document (DOC), Office Open XML (OOXML, DOCX), Rich Text Format (RTF), OpenDocument Text (ODT), Web page (HTML) and plain text (TXT) formats. [Download][3] the free evaluation version today to see how Aspose.Words for JasperReports can work for you.

#### New Releases and Updates

1.  **Aspose.Slides for Java** v2.0.0 - Improved PPTX and PPT support along with bug fixes
2.  **Aspose.Slides for .NET** v4.0.0 - Rendering PPTX to images, optimized export of PPT to PDF, exporting PPT to XPS and multi-page TIFF images
3.  **Aspose.Pdf.Kit for Java** v2.4.1 - Replace text, extract text and encryption improved
4.  **Aspose.Pdf for Reporting Services** v1.6.0 - Table of contents, bookmarks & hyperlinks support for SSRS2008
5.  **Aspose.Pdf.Kit for .NET** v3.7.0 - Improved PdfViewer and PdfConverter classes and enhanced MakeBooklet, DecodePage and ImportXML methods

#### Migration from MS Office Automation to Aspose – Aspose.Network for .NET

![](https://forum.aspose.com/)In this month’s newsletter we include information for migrating to [Aspose.Network for .NET][4] your Microsoft Office Automation code for getting list of messages from inbox folder of Microsoft Exchange Server mailbox. You can find the detailed article with complete code here.

#### Technical Article – Save/retrieve barcode images to/from MySQL database

In this article, we will use [Aspose.BarCode for Java][5] to generate barcode image and save it in MySQL database. We will then show you how to retrieve the barcode image from database and recognize the codetext from it. You can find the complete article here.

#### Tutorial Video – Print barcode on JasperReports and export to PDF

In this video tutorial, we will demonstrate how you can design a simple report using Netbeans IDE and iReport plug-in and add a simple image to the report. We will then use [Aspose.BarCode for JasperReports][6] to render a barcode image on the report during the runtime and export the report to PDF format for viewing and printing. You can view the video here.
