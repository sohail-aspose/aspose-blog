![](https://blog.aspose.com/wp-content/uploads/sites/2/2015/12/aspose_slides-for-java11.png "Aspose.Slides for Java")[![][1]](http://www.jython.org/)We are pleased to announce “[Aspose.Slides Java for Jython][2]”, a new project for Jython developers. Project is aimed to provide useful examples for [Jython][3] developers who want to utilize Aspose.Slides for Java API in their Jython applications to deal with Presentation documents.
