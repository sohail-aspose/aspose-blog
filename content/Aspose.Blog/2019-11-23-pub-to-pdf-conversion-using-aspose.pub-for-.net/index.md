PUB (Publisher) file format can now be converted into PDF using Aspose.PUB for .NET. Yes, you got it right. As shared in a [pre-release blog post][1] that Aspose was going to launch a new product to deal with PUB file format - Well, the wait is over!! Aspose.PUB for .NET is finally launched and available for download to be used in your .NET applications. Like every product by Aspose, Aspose.PUB has been released with fully loaded functionality to perform conversion of PUB to PDF format. The following details are very important to be gone through in order to set up and use the API to implement the offered functionality. Let’s see how it can be your next friendly interface which you will be using in your .NET application.

## Working with Aspose.PUB for .NET

Like other Aspose APIs, Aspose.PUB offers a complete user guide to set it up and use it. Documentation of the API gives comprehensive material and barely lacks any important detail which is significant to use it. From installation to implementation of the conversion functionality, it is all given in public user guide. Following are some important links you would like to visit:  

*   [System Requirements][2]
*   [Installation][3]
*   [Licensing][4]
*   [How to Run Examples][5]
*   [Developer Guide][6]

It is also recommended to visit [release notes][7] of [Aspose.PUB for .NET 19.10][8] before downloading and using it to have a detailed overview of what it offers in the available release.  

Work is not over yet. Yes, further developments are also underway to make the feature set of the API richer and incorporate more usable functionalities. Like always, API [Support Forum][9] is open for your suggestions and required functionalities. Don’t forget to [subscribe][10] Aspose blog for the latest updates about APIs.
