## C# API to Create YouTube Video Downloader {#section1}

[Aspose.HTML for .NET][4] API supports web scrapping, which is used to extract data from websites. Configuring web scrappers is usually quite a task in general, but you can perform such tasks easily with the API. You need to download the DLL files from the [New Releases][5] section, or configure it with the following [NuGet][6] installation command:

```
PM> Install-Package Aspose.HTML
```

## Create YouTube Video Downloader in C# {#section2}

You can create a YouTube video downloader with the following steps:

*   Specify the URL of the video to download.
*   Initialize an instance of the [MultimediaScraper][7] class.
*   Create a [Multimedia][8] object and check if VideoInfo is [YouTubeVideoInfo][9].
